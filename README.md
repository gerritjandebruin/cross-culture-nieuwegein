De website kan als volgt aangepast worden.

Maak een account op Github aan en log in. 
Ga naar https://github.com/gerritjandebruin/cross-culture-nieuwegein en druk op `.`.

Pas de volgende pagina's aan:
- _data/activiteiten-nl.yml voor de activiteiten
- _data/contacts.yml voor de contactpersonen
- _data/navigation-en.yml voor de Engelstalige navigatie
- _data/navigation-nl.yml voor de Nederlandstalige navigatie
- _data/partners-en.yml voor de Engelstalige partners
- _data/partners-nl.yml voor de Nederlandstalige partners
- _data/translations.yml voor sommige vertalingen
- en/index.md voor de Engelstalige homepage
- en/activities.md voor de Engelstalige activiteitenpagina
- en/contact.html voor de Engelstalige contactpagina
- en/partners.html voor de Engelstalige partnerpagina (zie ook _data/partners-en.yml)
- en/who-we-are.md voor de Engelstalige wie-zijn-wij pagina
- index.md voor de Nederlandstalige homepage
- activiteiten.md voor de Nederlandstalige activiteitenpagina (zie ook _data/activiteiten-nl.yml)
- contact.html voor de Nederlandstalige contactpagina (zie ook _data/contacts.yml)
- partners.html voor de Nederlandstalige partnerpagina (zie ook _data/partners-nl.yml)
- wie-zijn-wij.md voor de Nederlandstalige wie-zijn-wij pagina
